# Rabbitmq使用笔记

###### gist:

publish -> exchange(fanout, direct, topic) -> queue -> consumer

四种类型交换机：fanout 广播，direct 直连，topic主题，header 头交换机

fanout ，direct 绑定多个或单个 queueLabel

binding 过程通过 routeKey 来唯一识别
